# controler-your-game
Controller Game menggunakan python

Program ini memungkinkan kamu untuk mengendalikan game menggunakan tangan dengan bantuan kamera.
di saat anda menjalankan program nya anda sudah berada di game dengan program yg berjalan
 

---

## 🚀 Cara Penggunaan

1. Clone repository:
  
   git clone https://github.com/Ainun-Nadjib17/controler-your-game

   cd controller
   
3. pip install -r requirements.txt
4. python controller.py
5. mengontrol game:
jari 1 -> jari 2 <- jari 3 jump jari 4 slide
